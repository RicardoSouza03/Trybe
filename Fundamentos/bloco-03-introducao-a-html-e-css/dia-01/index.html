Nesta primeira aula sobre HTML, irei fazer meu primeiro projeto seguindo algumas especificações.

Estas são:

  1. Fazer uma lista de quem eu ja conversei na Trybe
  2. Colocar uma imagem externa
  3. Colocar uma imagem local.
  4. Fazer um paragrafo para a lista de pessoas.
  5. Linkar os paragros e as pessoas.
  6. Colocar um link para outro site.
